# AI Finance Tracker

개인 자산 관리와 가계부를 위한 웹 애플리케이션입니다.

## 주요 기능

- 자산 관리
  - 총 자산 현황 확인
  - 자산 유형별 관리 (예금, 적금, 투자, 보험 등)
  - 자산 상세 정보 확인

- 가계부
  - 수입/지출 내역 관리
  - 거래 추가 및 상세 보기
  - 월별 수입/지출 요약

## 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
- Headless UI

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```
